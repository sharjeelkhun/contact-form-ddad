== Description ==

Just another contact form plugin. Advance dynamic drag & drop, easy to use, flexible.

== Installation ==

1. Upload the `contact-form-ddad` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin in the 'Contact Form DDAD' settings page.

== Frequently Asked Questions ==

= How do I create a contact form using this plugin? =

To create a contact form, follow these steps:
1. Go to the 'Contact Form DDAD' settings page in your WordPress admin panel.
2. Click the 'Add New Form' button.
3. Drag and drop form elements onto your form.
4. Configure the form settings and notifications.
5. Save your form and copy the shortcode to add it to a post or page.
